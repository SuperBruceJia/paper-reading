# Super-PyTorch

This is the my own PyTorch Edition (1.8.0) that is optimized and built from source.

## Install on macOS

1. Intell Python Environment

```
$ conda create --name pytorch python=3.6 numpy scipy
$ conda activate pytorch
```

3. Install PyTorch

```
$ git clone --recursive https://github.com/pytorch/pytorch
$ cd pytorch

# if you are updating an existing checkout
$ git submodule sync
$ git submodule update --init --recursive
```

then

```
$ export CMAKE_PREFIX_PATH=${CONDA_PREFIX:-"$(dirname $(which conda))/../"}
$ MACOSX_DEPLOYMENT_TARGET=11.0 CC=clang CXX=clang++ python setup.py install
```
